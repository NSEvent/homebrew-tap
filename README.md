# homebrew-tap

Homebrew tap for [Kevin Tang](https://www.kevintang.xyz)'s apps.

## ControllerKeys

Map game controllers to keyboard, mouse, scrolling, and macros on macOS.

```sh
brew install --cask nsevent/tap/controllerkeys
```

Free 14-day trial — a license (from [Gumroad](https://thekevintang.gumroad.com/l/xbox-controller-mapper)) unlocks continued use, entered in **Settings → General**. More at <https://www.kevintang.xyz/apps/controller-keys>.
